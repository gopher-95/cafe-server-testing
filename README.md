# 🧪 Тестирование сервера «Поиск кафе»

Модульные тесты для HTTP-сервера поиска кафе с проверкой параметров `count` и `search`.

## 🔧 Что тестируется

- Корректная работа параметра `count` (ограничение количества результатов)
- Корректная работа параметра `search` (поиск по подстроке без учёта регистра)
- Обработка негативных сценариев

## 📦 Состав

| Файл | Назначение |
|------|------------|
| `main.go` | Код HTTP-сервера |
| `main_test.go` | Тесты сервера |

## 🧪 Тестовые функции

- `TestCafeWhenOk()` — позитивные сценарии
- `TestCafeNegative()` — негативные сценарии
- `TestCafeCount()` — проверка параметра `count`
- `TestCafeSearch()` — проверка параметра `search`

## 🛠️ Стек

- Golang
- Стандартная библиотека (`net/http/httptest`, `testing`)
- testify (`assert`, `require`)

## 🚀 Запуск тестов

```bash
go test .
